# Автоматизированная система обработки корпоративной почты

Хакатон, «Технологии программирования», НИУ ВШЭ, 2025–2026

## Что делает

Скрипт читает папку с входящими письмами, определяет категорию каждого и раскладывает файлы по папкам. В конце выводит статистику: сколько писем куда попало.

Категории: Инциденты, Доступ, ПО, Оборудование, Фин. вопросы, Юр. вопросы, HR, Встречи, Спам, Исходящие, Черновик, Прочее.

Если письмо не попало ни в одну категорию — идёт в «Прочее». Если файл не удалось прочитать — записывается в `output/trash.txt`.

## Структура

```
HackathonTP/
├── email_parser_and_classifier.py
├── run.sh
├── requirements.txt
├── tests/
└── README.md
```

## Как запустить

Открыть Git Bash, перейти в папку проекта и выполнить:

```bash
bash run.sh inbox
```

`inbox` — папка с письмами. Можно указать другой путь.

Перед первым запуском установить зависимости:

```bash
pip install -r requirements.txt
```

Результаты появятся в папке `output/`.
